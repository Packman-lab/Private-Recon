## Requirements
1. Python3 
   (https://www.python.org/ftp/python/3.8.5/Python-3.8.5.tar.xz)

2. Idle (optional)
    ```sh
    sudo apt install idle
    ```
3. Python Modules
    * 
    ```sh
    pip3 install os
    pip3 install socket
    pip3 install requests
    pip3 install hashlib
    pip3 install argparse
    pip3 install json
    pip3 install logging
    pip3 install time
    pip3 install nmap
    pip3 install sys
    ```
    
## HowToRun : 

#### On linux/mac
```sh
python3 recon_v1.py
```
#### On windows
```sh
python recon_v1.py
```
